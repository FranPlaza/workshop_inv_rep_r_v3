## Módulos
### Sesión I

  - Ventajas y dificultades
  - Etapas de una investigación reproducible
  - Ciencia abierta
  - Software necesarios
- **Módulo II**: Control de versiones
  - Motivaciones para el uso de control de versiones
  - Git/Github
  - Conceptos básicos
